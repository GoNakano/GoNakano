## Projects

### VR Glider Simulator

研究室のチーム開発プロジェクトとして、アブローラーとIMUセンサーを用いたVRグライダーシミュレータを開発しています。  
体験者の姿勢入力をもとにVR空間内のグライダーを操作し、リングやチェックポイントを通過しながらゴールを目指す体験型システムです。

主に以下を担当しています。

- メインゲームシーン
- ゲームマネージャー
- UI / HUD
- リング・チェックポイント・ゴール処理
- リザルト表示
- サウンド制御の枠組み
- TCP通信による外部機器連携
- パフォーマンス改善
- Git / GitHubを用いたチーム開発統合

※研究室のPrivateプロジェクトのため、コードは公開していません。
